📌 Overview

The Bank Loan Analytics Dashboard is an interactive Power BI dashboard designed to analyze loan data and provide insights into borrower profiles, loan distribution, credit scores, education levels, and loan status.

The dashboard enables users to monitor important banking KPIs and identify trends that support better lending decisions.

Insights
Education loans contribute the highest loan amount.
Male borrowers account for a slightly larger share of loans.
Mortgage owners have the highest average credit score.
Doctorate holders have the highest average loan interest rate.
Fair credit score category contains the largest number of borrowers.
Loan status is nearly evenly split between defaulters and non-defaulters.
